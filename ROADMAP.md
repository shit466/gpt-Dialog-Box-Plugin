# Roadmap

## 0.1 Prototype

- Inline branch panel beside AI answers
- Sticky branch index
- Keep, close, and delete branch actions
- Local-only persistence

## 0.2 Usability

- Editable branch titles
- Better mobile layout
- Export branches as Markdown
- Clear all branches action

## 0.3 Model Integration

- Options page for OpenAI API key and model
- Real branch replies using the original answer as context
- Streaming branch responses
- Error and retry states

## 0.4 Reliability

- More robust ChatGPT message detection
- Stable anchor migration when content changes
- `chrome.storage.local` persistence
- Automated browser smoke tests

## Later

- Support Claude, Gemini, Perplexity, and other chat products
- Branch search
- Branch graph view
- Optional sync backend
