# Contributing

Thanks for considering a contribution.

## Local Setup

This prototype does not require a build step.

```bash
npm test
```

Then load the project folder in Chrome through `chrome://extensions`.

## Pull Request Guidelines

- Keep changes focused and easy to review.
- Test on `chatgpt.com` after UI or selector changes.
- Update the README when behavior or setup changes.
- Avoid collecting or sending chat content without explicit user control.

## Useful Areas

- More stable message anchoring
- Real model integration
- Options page
- Branch search/export
- Adapters for additional AI chat products
